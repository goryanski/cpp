Создать приложение "Автосалон"

Создайте список машин, которые состоят из:
- марка
- модель
- цвет
- цена
- год выпуска
- страна производитель


Реализовать функционал:
- поиск машины по заданному критерию
- вывод машин:
	- по цвету
	- по году выпуска
	- по стране производителя
	- по цене (от 0 до указанной цены)
- поиск самой дешевой машины
- поиск самой дорогой машины


Реализовать меню для взаимодействия с пользователем
